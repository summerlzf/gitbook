# 财商（FQ：Financial Quotient）

财商，又称：金融智商，主要指个人或者集体认识、创造和管理财富的能力，包括观念、知识、行为等方面

#### 凡是涉及到跟钱有关的安排，都是理财



#### 工资收入：自己付出时间、精力、汗水挣来的钱

#### 非工资收入：不需要去劳动，也能产生的收入，比如投资、出书的版税等



#### 财务自由，就是你的<u>非工资收入</u>大于生活总支出



### 资产

资产的内涵是现金流，根据产生的现金流不同，可以划分为3种：

- **生钱资产**：能给你持续带来净现金流入的东西
- **耗钱资产**：能给你持续带来净现金流出的东西
- **其他资产**：产生的净现金流为0的东西

### 支出

- **好支出**：可以带来更多收入的支出，例如购买【生钱资产】、花钱学习英语、理财等技能
- **坏支出**：让你的钱变得更少的支出，例如购买【耗钱资产】、非必要性消费、乱投资导致的亏损等



### 富人的两个秘密

1. 生钱资产占总资产80%以上
2. 好支出占总支出80%以上

### 复利三要素

- 本金
- 时间
- 收益率

复利公式：最终收益 = 本金 * (1 + 收益率) ^ 时间

其中，最重要的是 **收益率**



### 最重要的理财工具

低风险、高收益，年化收益率一般在15%-30%，主要包括：

- 生钱资产类型的股票
- REITs
- 房地产

一定要**摒弃**的一种理财思维：~~低风险=低收益，高风险=高收益~~

【生钱资产类型的股票】就是指 **持续分红的股票**（可通过查看股票的 股息率 判断收益率）

【REITs】指房地产投资信托基金，一般的REITs都是持续高分红的，因为各国法律都规定REITs每年至少要把90%以上的净利润分配给投资者

【可转债】可以转换成股票的债券，年化收益率可达到 10%-20%

【股票指数基金】复制股票指数的股票基金，年化收益率一般在 10%-15%

#### 高风险的理财工具

- P2P：本质是互联网放高利贷，不在国家交易所交易的标准化理财工具
- 私募基金：面向少数人募集成立的基金，非标准化的理财工具
- 比特币：数字货币，不在国家交易所交易的标准化理财工具，本身不产生现金流，只能赚差价（央行刚发行的数字货币除外，央行的数字货币没有投资价值，只是单纯用来交易的）
- 耗钱资产类型的房地产：带来持续的净现金流出，只能靠未来房价上涨赚钱，具有很大的不确定性
- 期货：在期货交易所交易的标准化合约，没有信用风险，但期货本身不产生现金流，靠未来的差价赚钱，而且期货一般都有高杠杆，容易亏完本金

**注意：一定要远离这些高风险的理财工具**



#### 实现财务自由的三大核心工具：企业、股票、REITs

- 购买生钱资产，主要是购买生钱资产类型的股票和REITs
- 创造生钱资产，主要就是创造企业



### 资产负债率

**资产负债率 = 总负债 ÷ 总资产 × 100%**

- 资产负债率 > 70%，有比较大的偿债风险
- 资产负债率 < 40%，几乎没有偿债风险



### 反映企业竞争力的指标

- 应付（√）——货已经用了，但是钱还没有给供应商
- 预收（√）——先收了经销商的钱，但是货还没有给
- 应收（×）——货已经被经销商拉走了，但还没收到钱
- 预付（×）——钱已经给供应商了，但是货还没有收到

√：表示企业具有正向竞争力

×：表示企业具有负向竞争力

【应付、预收】越大越牛

【应收、预付】越大越弱



### 关于股票

正确理财的人，是根本不用花时间关注估价波动的

只要买到了好企业（真实利润在增长），估价跌得越多，长期收益率就越高

公司的真实利润减少才是真正的风险

价值投资 = 好公司 + 好价格

#### 投资股票的正确方法：

1. 选出内在价值高的好企业
2. 在价格大幅低于内在价值的时候，及时买入

### 关于企业/公司

- 【好公司】的股价持续大爆跌你会赚大钱
- 【垃圾公司】的股价持续大爆跌你会亏大钱
- 股价持续大爆跌不是风险，【公司利润】持续减少才是风险
- 【财报分析】可以看出一个公司的过去和现在
- 【企业分析】可以看出一个公司的现在和未来
- 投资股票就是投资企业
- 投资股权也是投资企业
- 投资债券（企业债、可转债），还是投资企业
- 投资REITs也可以看作投资企业，分析REITs的财务报表和分析企业的财务报表都是一样的



### 各类人群和企业的关系

- E：拥有一份工作——雇员，打工者
- S：做自己的老板——自由职业者，如：包子店主、水果摊主、工作室老板
- B：建立一个别人为你工作的系统——企业家，集团性的企业，如：阿里、腾讯
- I：钱为你工作——投资家

B和S很类似，最大的区别在于：B的企业离开B的人照样可以运作得很好，S的企业离开S的人就会倒闭



### 企业分析工具——强大企业模型

【强大企业模型】包括9个关键要素：领导者、企业文化、治理结构、商业模式、团队、沟通、现金流、系统、法规

其中最核心的要素是：领导者；其他8个关键要素都和领导者有着直接或者间接的关系



### 护城河

护城河，也叫企业的核心竞争力，是竞争对手无法复制或模仿的竞争优势

护城河主要有四种：网络效应、转换成本、成本优势、独特资源

转换成本是指：客户从购买一个供应商的产品转向购买另一个供应商产品时所增加的费用

- 目前世界上70%以上的企业，这4种护城河一个都不具备

- 具有1种护城河的企业就有强大的生存能力和较好的盈利能力
- 同时具有2种护城河的企业可能是行业龙头公司
- 同时具备3种或以上护城河的企业，全世界不超过100家（具备4个的如：Facebook、伯克希尔哈撒韦）

投资企业，最好买具有3种或以上的企业，至少也要买具有2种或以上的企业



### 股票海选

爱问财，选股网站：http://www.iwencai.com

输入条件：

1. 2014年到2018年的ROE大于20%
2. 2014年到2018年的净利润现金含量大于80%

##### 概念科普

```
ROE（Return on Equity），净资产收益率，又称：股东权益报酬率、净值报酬率、净资产利润率等，是指净利润与平均股东权益的百分比，是公司税后利润除以净资产得到的百分比率，该指标反映股东权益的收益水平
```

一般要选择ROE大于20%的公司

```
净利润现金含量，是指刨除掉应收账款，净利润里面有多少是实实在在拿到手的
```

一般会选择净利润现金含量大于100%的公司
