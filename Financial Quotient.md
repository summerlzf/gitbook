# 财商（FQ：Financial Quotient）

财商，又称：金融智商，主要指个人或者集体认识、创造和管理财富的能力，包括观念、知识、行为等方面

#### 凡是涉及到跟钱有关的安排，都是理财



#### 工资收入：自己付出时间、精力、汗水挣来的钱

#### 非工资收入：不需要去劳动，也能产生的收入，比如投资、出书的版税等



#### 财务自由，就是你的<u>非工资收入</u>大于生活总支出

### 实现财务自由的四大准则

- 正向现金流
- 制定投资标准
- 全球化配置
- 长期投资



### 资产

资产的内涵是现金流，根据产生的现金流不同，可以划分为3种：

- **生钱资产**：能给你持续带来净现金流入的东西
- **耗钱资产**：能给你持续带来净现金流出的东西
- **其他资产**：产生的净现金流为0的东西

### 支出

- **好支出**：可以带来更多收入的支出，例如购买【生钱资产】、花钱学习英语、理财等技能
- **坏支出**：让你的钱变得更少的支出，例如购买【耗钱资产】、非必要性消费、乱投资导致的亏损等



### 富人的两个秘密

1. 生钱资产占总资产80%以上
2. 好支出占总支出80%以上

### 复利三要素

- 本金
- 时间
- 收益率

复利公式：最终收益 = 本金 * (1 + 收益率) ^ 时间

其中，最重要的是 **收益率**



### 最重要的理财工具

低风险、高收益，年化收益率一般在15%-30%，主要包括：

- 生钱资产类型的股票
- REITs
- 房地产

一定要**摒弃**的一种理财思维：~~低风险=低收益，高风险=高收益~~

【生钱资产类型的股票】就是指 **持续分红的股票**（可通过查看股票的 股息率 判断收益率）

【REITs】指房地产投资信托基金，一般的REITs都是持续高分红的，因为各国法律都规定REITs每年至少要把90%以上的净利润分配给投资者

【可转债】可以转换成股票的债券，年化收益率可达到 10%-20%

【股票指数基金】复制股票指数的股票基金，年化收益率一般在 10%-15%

#### 高风险的理财工具

- P2P：本质是互联网放高利贷，不在国家交易所交易的标准化理财工具
- 私募基金：面向少数人募集成立的基金，非标准化的理财工具
- 比特币：数字货币，不在国家交易所交易的标准化理财工具，本身不产生现金流，只能赚差价（央行刚发行的数字货币除外，央行的数字货币没有投资价值，只是单纯用来交易的）
- 耗钱资产类型的房地产：带来持续的净现金流出，只能靠未来房价上涨赚钱，具有很大的不确定性
- 期货：在期货交易所交易的标准化合约，没有信用风险，但期货本身不产生现金流，靠未来的差价赚钱，而且期货一般都有高杠杆，容易亏完本金

**注意：一定要远离这些高风险的理财工具**

### 资产负债率

**资产负债率 = 总负债 ÷ 总资产 × 100%**

- 资产负债率 > 70%，有比较大的偿债风险
- 资产负债率 < 40%，几乎没有偿债风险

```
资产负债率70%，说明每100元的资产中，就有70元是借来的，剩下的30元才是自己的
```



### 反映企业竞争力的指标

- 应付（√）——货已经用了，但是钱还没有给供应商
- 预收（√）——先收了经销商的钱，但是货还没有给
- 应收（×）——货已经被经销商拉走了，但还没收到钱
- 预付（×）——钱已经给供应商了，但是货还没有收到

√：表示企业具有正向竞争力

×：表示企业具有负向竞争力

【应付、预收】越大越牛

【应收、预付】越大越弱

### 护城河

护城河，也叫企业的核心竞争力，是竞争对手无法复制或模仿的竞争优势

护城河主要有四种：网络效应、转换成本、成本优势、独特资源

转换成本是指：客户从购买一个供应商的产品转向购买另一个供应商产品时所增加的费用

- 目前世界上70%以上的企业，这4种护城河一个都不具备

- 具有1种护城河的企业就有强大的生存能力和较好的盈利能力
- 同时具有2种护城河的企业可能是行业龙头公司
- 同时具备3种或以上护城河的企业，全世界不超过100家（具备4个的如：Facebook、伯克希尔哈撒韦）

投资企业，最好买具有3种或以上的企业，至少也要买具有2种或以上的企业

##### 概念科普

```
ROE（Return on Equity），净资产收益率，又称：股东权益报酬率、净值报酬率、净资产利润率等，是指净利润与平均股东权益的百分比，是公司税后利润除以净资产得到的百分比率，该指标反映股东权益的收益水平
简单来说，ROE=20%，说明我给这家公司投了100元，他能给我赚20元
```

一般要选择ROE大于20%的公司

```
净利润现金含量，是指刨除掉应收账款，净利润里面有多少是实实在在拿到手的
```

一般会选择净利润现金含量大于100%的公司

```
毛利率=10%，说明公司每销售100元，能够赚10元
```

#### 股票/企业分析的关键指标

以雪球APP为例，观察和分析以下几个指标

- 行业对比：可以初步判断该企业处于行业的哪个水平
- 资产负债率：可判断企业的偿债风险
- ROE：收益率的一个直观展示
- 毛利率：反映利润情况，毛利率高，说明企业的利润高
- 净利润和经营现金流：企业赚的钱和实际收到的钱，间接反映企业在行业中的地位
- 分红：一般而言，企业只有在赚到钱的情况下，才会分红，分红趋势的变化，也大致反映企业的发展变化



### 市盈率

市价盈利比率，是一个投资回报指标，即：市场投资者为了获得1元的盈利，愿意付出的价格

一般来说，用15元的价格获得1元的盈利是相对合理的，所以，市盈率为15的企业（股票），年化收益率就是：1÷15=6.67%，市盈率越高，年化收益率越低，反之亦然

根据市盈率大小，可设置的不同的定投金额

- 市盈率小于15时，每月定投X元
- 市盈率小于12时，每月定投2X元
- 市盈率小于10时，每月定投4X元

注意：市盈率在30或者10都是不太合理的

```
深证市盈率走势图：http://www.legulegu.com/stockdata/shenzhenPE
```

在深证过去20年的市盈率走势图可以看到，市盈率到50以上，就该往下跌了，说明被高估了；如果是20以下，说明被低估了，该反弹了

#### 根据市场的市盈率定投指数基金

- 0 < 市盈率 < 20，定投 4X
- 20 < 市盈率 < 25，定投 3X
- 25 < 市盈率 < 30，定投 2X
- 30 < 市盈率 < 50，持有
- 市盈率 > 50，卖出



### REITs

REITs（Real Estate Investment Trusts，房地产投资信托基金），是一种**依照法律程序**成立的，以发行股票或基金单位的方式汇集投资者的资金，由**专业机构**进行房地产的经营管理，并将**90%以上的税后收入净额**分配给投资者的一种信托组织

#### REITs的特点

1. 定期强制高分红
2. 投资门槛低，一般几千元就可以投
3. 交易成本低
4. 流动性强
5. 专业化管理
6. 收益率高

美国REITs每个季度派息一次，香港REITs每半年派息一次

一般REITs会免企业所得税，投资者买卖REITs只需支付佣金，和买卖股票一样，不用支付其他高额税费

REITs的管理团队都是房地产领域的专业人士，他们管理物业的能力一般比个人强很多

中国内地目前没有REITs，所以需要从港美股券商那里购买，例如：富途证券

#### REITs的风险

REITs的本质就是房地产，房地产有的风险REITs基本都有



### 基金

分为【主动型基金】和【被动型基金】

【主动型基金】有基金经理决定买的那些基金

【被动型基金】就是【指数基金】，只会根据指数配置股票，不会人为选择

80%的主动型基金是跑不赢指数基金的

【指数基金】会自动淘汰那些不好的股票，加入满足条件的股票，特点是：永生不死、长期上涨



### 通货膨胀

通货膨胀 = 通货 + 膨胀

通货，即通用货币、法定货币

#### 费雪方程式：MV = PT

- M：货币量
- V：货币流通速度
- P：价格水平
- T：交易的商品总量

在正常社会活动种，V和T一般是比较稳定的，所以，货币量M决定物价P，M和P呈正相关，**货币越来越多，物价也就越来越高**，钱也就越来越不值钱了，这就是通货膨胀的根源

```
CPI，Consumer Price Index，消费者物价指数
```

实际CPI = M2增长率 - GDP增长率

- M2增长率反映一个国家每年钱的增长速度
- GDP增长率反映一个国家每年创造的商品和服务的增长速度

实际CPI，其实就是通货膨胀率

中国过去20年（1998—2018）的实际年化通货膨胀率为 10.04%，也就是过去20年平均每年货币贬值10%



【投资】以赚取现金流为主，赚差价为辅，比如购买高股息率的好股票、高分红的REITs

【投机】以赚取差价为主，投机的实质，其实就是赌博
